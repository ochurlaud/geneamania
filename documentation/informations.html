Informations

Une fenêtre contenant des informations générales sur GénéGraphe est accessible en cliquant sur le bouton « ? » de la barre de menus.

GénéGraphe vous affiche la fenêtre suivante :

Vous y trouvez toutes les informations sur la version du logiciel (numéro de version, date de génération et compatibilité avec Généamania).

Sur 4 lignes, vous avez des indications sur la gestion de la mémoire :

  • la mémoire maximale disponible pour le logiciel ;
  • la taille de la partition mémoire actuellement utilisée ;
  • la mémoire libre dans la partition ;
  • la mémoire utilisée dans la partition…

Ensuite, vous trouvez le nom et la version de votre système d'exploitation ainsi que la référence de l'environnement Java installé.

Vous trouvez l'adresse mail que vous pouvez utiliser pour contacter le développeur de GénéGraphe et, pour finir, comment accéder à la documentation de GénéGraphe sur votre ordinateur.

Ces renseignements peuvent servir en cas de problème, pour renseigner le développeur sur votre environnement de travail.

Les renseignements sur la mémoire sont pris au moment ou vous appelez cette fenêtre. Vous pouvez ainsi savoir comment évolue la mémoire disponible en fonction de votre travail.
